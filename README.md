# NaN — Notes and Notions

NaN stands for “Notes and Notions”.

The NaN (Notes and Notions) project contains public-facing documentation for
AI/WB and acts as an exploration space for the project.

NaN is developed using the conventions and workflows defined by AI/WB.
Use “AI/WB” in prose unless specifically referring to the repository name
`AI_WB`.

AI/WB repository: https://github.com/RichMorin/AI_WB
